CascadesDB Upload

To upload data to CascadesDB, first ensure that you have the required metadata and that the data you intend to upload is compressed (using bzip2 or gzip) into a single archive file. Multiple simulations can be uploaded in a single archive but must differ only in the PKA recoil direction if so.

The archive must contain:

  • .xyz output files
  • .xyz input file, if the initial state of the material is other than a perfect crystal

The archive should contain:

  • the MD code input files corresponding to each .xyz output
  • A file or URL to the interatomic potential used in the simulation

The archive may contain:

  • A text readme file
  • PDF copies of any publication(s) relating to the data
